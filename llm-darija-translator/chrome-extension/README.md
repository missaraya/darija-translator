# Chrome Extension — Darija Translator

Manifest V3 · chrome.sidePanel · chrome.contextMenus

---

## Features

- Right-click any selected text → "Translate to Darija"
- Side panel opens automatically with the selected text pre-filled
- One-click translation via the backend REST API
- Copy translation to clipboard
- Read aloud (Text-to-Speech) with Arabic voice
- Settings panel to configure backend URL and credentials
- Works from the toolbar icon too (opens side panel directly)

---

## Prerequisites

- Google Chrome 116+ (Manifest V3 side panel support)
- The backend server must be running on `http://localhost:8080`
  (see `backend/README.md`)

---

## Setup Steps

### 1. Generate Icons

```bash
cd chrome-extension/icons
pip install Pillow
python generate_icons.py
```

This creates `icon16.png`, `icon48.png`, and `icon128.png`.

### 2. Load the Extension in Chrome

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **"Load unpacked"**
4. Select the `chrome-extension/` folder
5. The "Darija Translator" extension appears in your extensions list

### 3. Configure the Extension (First Time)

1. Click the 🌙 extension icon in the Chrome toolbar
2. The side panel opens — click ⚙️ (Settings)
3. Verify:
   - Backend URL: `http://localhost:8080`
   - Username: `student`
   - Password: `student123`
4. Click **Save Settings**

---

## Usage

### Option 1 — Context Menu (recommended demo flow)
1. Visit any webpage
2. Select some English text
3. Right-click → **"Translate to Darija"**
4. The side panel opens with the selected text pre-filled
5. Click **Translate**

### Option 2 — Toolbar Button
1. Click the 🌙 icon in the Chrome toolbar
2. Type text manually in the side panel
3. Click **Translate** (or press `Ctrl+Enter`)
