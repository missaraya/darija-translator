# Assets

Place the following Expo asset files here:

- `icon.png` — 1024×1024 app icon
- `splash.png` — 1242×2436 splash screen image
- `adaptive-icon.png` — 1024×1024 adaptive icon foreground (Android)
- `favicon.png` — 32×32 web favicon

These are standard Expo assets. You can generate them with:
```bash
npx expo install expo-asset
```
Or use any PNG files of the correct dimensions.

For the demo, Expo will use default placeholders if these files are missing.
